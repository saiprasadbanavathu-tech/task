# task 2 This project focuses on building and optimizing machine learning models for house price prediction using the California Housing Dataset. It covers the complete ML pipeline, including data preprocessing, feature engineering, model training, hyperparameter tuning, and performance evaluation.

By comparing multiple regression algorithms, the project identifies the most accurate model for predicting housing prices and demonstrates how optimization techniques can improve predictive performance.

🚀 Key Highlights
Data Cleaning & Preprocessing
Feature Engineering
Feature Scaling with StandardScaler
Model Training & Evaluation
Hyperparameter Tuning (GridSearchCV)
Performance Comparison of Multiple Models
Data Visualization & Insights
